# Zilliz

>[Zilliz Cloud](https://zilliz.com/doc/quick_start) 是一个针对 `LF AI Milvus®` 的完全托管的云服务，

## 安装与设置

安装 Python SDK：
```bash
pip install pymilvus
```

## 向量存储

Zilliz 索引的封装器使您可以将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_community.vectorstores import Milvus
```

有关 Milvus 封装器的更详细说明，请参阅 [此笔记本](/docs/integrations/vectorstores/zilliz)
