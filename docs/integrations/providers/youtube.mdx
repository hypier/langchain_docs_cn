# YouTube

>[YouTube](https://www.youtube.com/) 是由谷歌提供的在线视频分享和社交媒体平台。
> 我们下载 `YouTube` 的转录文本和视频信息。

## 安装与设置

```bash
pip install youtube-transcript-api
pip install pytube
```
查看[使用示例](/docs/integrations/document_loaders/youtube_transcript)。
