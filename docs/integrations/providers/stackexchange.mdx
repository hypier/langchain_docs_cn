# Stack Exchange

>[Stack Exchange](https://en.wikipedia.org/wiki/Stack_Exchange) 是一个涵盖多个领域的问答(Q&A)网站网络，每个网站专注于特定主题，用户的提问、回答和互动都受到声望奖励机制的影响。

本页面介绍如何在 LangChain 中使用 `Stack Exchange API`。

## 安装和设置
- 使用以下命令安装依赖项 
```bash
pip install stackapi
```

## 包装器

### 实用工具

存在一个 StackExchangeAPIWrapper 实用工具，它封装了这个 API。要导入这个实用工具：

```python
from langchain_community.utilities import StackExchangeAPIWrapper
```

有关此封装的更详细说明，请参见 [此笔记本](/docs/integrations/tools/stackexchange)。

### 工具

您也可以轻松地将此包装器加载为工具（与代理一起使用）。
您可以通过以下方式做到这一点：
```python
from langchain.agents import load_tools
tools = load_tools(["stackexchange"])
```

有关工具的更多信息，请参见[此页面](/docs/how_to/tools_builtin)。
