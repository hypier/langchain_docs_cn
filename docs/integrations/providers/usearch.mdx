# USearch
>[USearch](https://unum-cloud.github.io/usearch/) 是一个更小、更快的单文件向量搜索引擎。

>`USearch` 的基本功能与 `FAISS` 相同，如果您曾经研究过近似最近邻搜索，界面应该会很熟悉。 
> `USearch` 和 `FAISS` 都采用 `HNSW` 算法，但在设计原则上有显著不同。 `USearch` 紧凑且广泛兼容 FAISS， 
> 在不牺牲性能的情况下，主要关注用户定义的度量和更少的依赖。

## 安装和设置

我们需要安装 `usearch` python 包。

```bash
pip install usearch
```
