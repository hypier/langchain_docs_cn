# 01.AI

>[01.AI](https://www.lingyiwanwu.com/en) 是由李开复博士创立的全球公司，处于 AI 2.0 的前沿。他们提供前沿的大型语言模型，包括 Yi 系列，参数范围从 6B 到数百亿。01.AI 还提供多模态模型、开放 API 平台，以及开源选项，如 Yi-34B/9B/6B 和 Yi-VL。

## 安装和设置

在中国站点 [这里](https://platform.lingyiwanwu.com/apikeys) 或全球站点 [这里](https://platform.01.ai/apikeys) 注册并获取 API 密钥。
