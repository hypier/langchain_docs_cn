# Trello

>[Trello](https://www.atlassian.com/software/trello) 是一个基于网页的项目管理和协作工具，允许个人和团队组织和跟踪他们的任务和项目。它提供了一个称为“看板”的可视化界面，用户可以在其中创建列表和卡片以表示他们的任务和活动。
>TrelloLoader 允许我们从 `Trello` 看板加载卡片。

## 安装与设置

```bash
pip install py-trello beautifulsoup4
```

请参阅 [设置说明](/docs/integrations/document_loaders/trello)。
