# 维基百科

>[维基百科](https://wikipedia.org/) 是一个由志愿者社区（称为维基人）通过开放协作和使用名为 MediaWiki 的基于维基的编辑系统编写和维护的多语言免费在线百科全书。 `Wikipedia` 是历史上最大、阅读量最高的参考作品。

## 安装与设置

```bash
pip install wikipedia
```

## 文档加载器

查看 [用法示例](/docs/integrations/document_loaders/wikipedia)。

```python
from langchain_community.document_loaders import WikipediaLoader
```

查看[使用示例](/docs/integrations/retrievers/wikipedia)。
